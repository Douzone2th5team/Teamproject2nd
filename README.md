## 기능 요구 사항

> 회원 관리
>
- 로그인/회원가입/회원탈퇴 기능
- 친구 등록 및 삭제 기능
- 마이페이지를 통해 프로필 수정 기능
- 

> 실시간 채팅
>
- 1:1채팅, 그룹채팅, 대화상대 추가 초대 기능
- 사진, 파일, 동영상 전송 기능
- 채팅방 내용 검색 기능
- 이모티콘 기능

> 게시판
>
- 게시물 등록/수정/삭제 기능
- 게시물 댓글 작성 기능
- 게시물 이미지, 첨부 파일 등록 기능
-
